## 7. 性能需求分析

| 性能点 | 分析 | 目标值 | 建议 |
|--------|------|--------|------|
| 接口响应时间（策略配置） | 含数据库写入，低频操作 | P99 < 200ms | 无需优化 |
| IP 提取延迟（GetClientIP） | 纯内存操作（CIDR 匹配 + 字符串解析） | P99 < 0.1ms | 无需优化 |
| IP 策略校验延迟 | O(n)，n ≤ 100 条 CIDR，每次解析 | P99 < 1ms | 如有压测瓶颈，增加 Token 级 CIDR 缓存（TTL=60s） |
| 可信代理匹配 | O(m)，m 为代理数量（通常 < 10） | P99 < 0.1ms | 无需优化 |
| 全局变量内存 | `trustedProxyCIDRs`，< 50 条 | < 1KB | 无需优化 |
