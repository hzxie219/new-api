## 模块功能介绍

### 核心模块

| 模块名称 | 功能描述 | 目录路径 | 相关技术 |
|---------|---------|---------|---------|
| router | HTTP 路由分发，将请求路由到 API、Relay、Dashboard 和 Web 处理器 | `new-api/router/` | Gin |
| controller | HTTP 请求处理器，负责参数校验、调用 service 层并返回响应 | `new-api/controller/` | Gin |
| service | 核心业务逻辑层，实现计费、配额、日志、任务调度等 | `new-api/service/` | Go |
| model | 数据模型与数据库访问层，使用 GORM 操作 SQLite/MySQL/PostgreSQL | `new-api/model/` | GORM |
| relay | AI API 中继/代理核心，负责请求转发、适配和响应处理 | `new-api/relay/` | Go |
| relay/channel | 各上游 AI 服务商适配器（OpenAI、Claude、Gemini、AWS 等 40 个） | `new-api/relay/channel/` | Go |
| web | React 前端管理后台，提供用户界面、管理控制台和 Playground | `new-api/web/` | React 18、Vite、Semi Design |

### 公共模块

| 模块名称 | 功能描述 | 目录路径 |
|---------|---------|---------|
| common | 共享工具库（JSON 封装、Redis、加密、邮件、HTTP 客户端、速率限制等） | `new-api/common/` |
| middleware | 中间件集合（认证、限流、CORS、日志、分发、Gzip 等） | `new-api/middleware/` |
| setting | 配置管理（比例设定、模型配置、运营配置、系统配置、性能配置） | `new-api/setting/` |
| dto | 数据传输对象，定义请求/响应结构体 | `new-api/dto/` |
| constant | 常量定义（API 类型、渠道类型、上下文 Key 等） | `new-api/constant/` |
| types | 类型定义（中继格式、文件来源、错误类型） | `new-api/types/` |
| i18n | 后端国际化（go-i18n，支持 en/zh） | `new-api/i18n/` |
| oauth | OAuth 服务商实现（GitHub、Discord、OIDC 等） | `new-api/oauth/` |
| logger | 日志系统 | `new-api/logger/` |
| pkg | 内部公共包（cachex 缓存、ionet 网络工具） | `new-api/pkg/` |
